# amd-experiments — TurboQuant & block KV compression on AMD MI300X

This repository contains **benchmarks**, **HIP/Triton kernels**, and **reports** for studying **KV cache compression** on **AMD Instinct MI300X (gfx942)**. The main comparison is **TurboQuant (TQ3/TQ4)** versus **block-diagonal rotations** ported from the RotorQuant ecosystem: **PlanarQuant**, **IsoQuant**, and **RotorQuant** (Triton on ROCm).

## What you get here

- **End-to-end decode / prefill / quality** scripts under `benchmarks/`.
- **Microbenchmarks**: compress/decompress bandwidth, attention microbenches, batch scaling.
- **HIP reference** for TurboQuant-style pack/unpack in `kernels/` (standalone binaries work on MI300X; Python+PyTorch uses the pure-PyTorch path where noted in `CLAUDE.md`).
- **Reports**: `report/final_report_v2.md` (multi-method) and `report/final_report.md` (earlier). Figures are generated by `report/generate_figures_v2.py` → `report/figures_v2/`.
- **Results**: JSON outputs in `results/` (git-tracked where useful; large sweeps can be regenerated).

## Quick start

```bash
cd amd-experiments

# Consolidate all benchmark JSON into a verbose text + optional merged JSON
python3 scripts/consolidate_benchmarks.py --verbose --write-json results/consolidated_benchmarks.json

# Regenerate v2 figures (needs matplotlib + results/*.json)
python3 report/generate_figures_v2.py

# Compress / decompress kernel microbench (same as §4 of the report)
python3 benchmarks/bench_compress_decompress.py --n-vectors 4096 --n-iters 50
```

Interactive notebook:

```bash
jupyter notebook notebooks/consolidate_benchmarks.ipynb
```
